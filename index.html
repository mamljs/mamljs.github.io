conventions

  1. pathname starts with and ends with /
    1. example: '/', '/download/', '/blog/'
    2. wrong: 'download' '/download', 'blog/'
  2. each page is a directory, not file
    1. example: '/download/', '/blog/'
    2. wrong: '/download.html'ï¼Œ '/blog.htm'
